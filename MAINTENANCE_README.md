## Come utilizzare

Questa pagina HTML pu√≤ essere deployata temporaneamente mentre si fanno modifiche al sito principale.

### Opzioni di deploy:

1. **Netlify/Vercel**: 
   - Rinomina `maintenance.html` in `index.html` temporaneamente
   - Oppure configura un redirect nella dashboard

2. **GitHub Pages**:
   - Rinomina `maintenance.html` in `index.html`
   - Pusha le modifiche

3. **Server manuale**:
   - Sostituisci temporaneamente l'`index.html` del sito con `maintenance.html`
   - Ricorda di fare backup del file originale

### Personalizzazione

- Modifica l'email di contatto nella sezione `contact-info`
- Personalizza i colori nelle variabili CSS `:root` e `[data-theme="dark"]`
- Modifica il logo "RZ" con il tuo nome completo se preferisci

### Ripristino

Quando hai finito le modifiche, ripristina il file `index.html` originale.
